# Breakpoints

## 概要

MUIテーマのブレークポイントシステム。kaze-uxに準拠したカスタム値を設定。

| ブレークポイント | 値     |
| :--------------- | :----- |
| mobile           | 0px    |
| tablet           | 768px  |
| laptop           | 1024px |
| desktop          | 1280px |
| wide             | 1536px |
