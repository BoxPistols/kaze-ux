## Usage

### MUI Components

```tsx
import { useTheme } from '@mui/material/styles'

function MyComponent() {
  const theme = useTheme()
  return (
    <Box
      sx={{
        backgroundColor: theme.palette.primary.main,
        color: theme.palette.primary.contrastText,
      }}>
      MUI Component
    </Box>
  )
}
```

### Tailwind CSS Classes

```tsx
function MyComponent() {
  return (
    <div className='bg-primary-main text-primary-foreground'>
      Tailwind Component
    </div>
  )
}
```

### Shared Usage

Both MUI and Tailwind use the same color tokens from `colorData`, ensuring design consistency across all components.
