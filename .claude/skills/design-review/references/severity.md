# 重大度ルール

| レベル       | 基準                                        | 例                       |
| ------------ | ------------------------------------------- | ------------------------ |
| **Critical** | アクセシビリティ違反・React.FC 使用・any 型 | A01, A03, C01, C06       |
| **Major**    | ハードコード色値・旧API使用・export default | K01, C02, C03            |
| **Minor**    | スペーシング・命名・AI生成パターン          | S01, AI01-04             |
| **Info**     | 推奨事項・改善提案                          | ベストプラクティスの提案 |

## 対応優先度

- **Critical**: 即時修正必須。マージブロック
- **Major**: PR 内で修正すべき
- **Minor**: 次回以降で対応可
- **Info**: 参考情報
