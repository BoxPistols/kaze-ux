# Kaze Design System — 禁止パターン

AI コード生成・手動実装の両方で準拠すべきルール。

## コンポーネント

| ID  | 禁止                                         | 理由                                 |
| --- | -------------------------------------------- | ------------------------------------ |
| C01 | `React.FC`, `FC`, `FunctionComponent` の使用 | plain function + typed props を使う  |
| C02 | `<Grid item xs={12}>` (旧API)                | MUI v7: `<Grid size={{ xs: 12 }}>`   |
| C03 | `export default`                             | `export const` で named export       |
| C04 | セミコロン                                   | Prettier 設定で省略                  |
| C05 | ダブルクォート（文字列）                     | シングルクォートを使用               |
| C06 | `any` 型                                     | strict mode。具体的な型を定義        |
| C07 | `window.confirm()` / `window.alert()`        | `ConfirmDialog` コンポーネントを使用 |
| C08 | MUI `Button` と CVA `Button` の混在          | ページ内でどちらか統一               |

## カラー

| ID  | 禁止                           | 代替                              |
| --- | ------------------------------ | --------------------------------- |
| K01 | `#0EADB8` 等のハードコード色値 | `primary.main` 等のトークン参照   |
| K02 | `text-black`, `text-white`     | `text.primary`, `common.white`    |
| K03 | `bg-gray-*` 直接指定           | `grey.100` 〜 `grey.900` トークン |
| K04 | `opacity` での色調整           | トークンの `lighter` バリアント   |

## タイポグラフィ

| ID  | 禁止                                | 代替                                      |
| --- | ----------------------------------- | ----------------------------------------- |
| T01 | `font-size` の px 直書き            | MUI typography variant (`body1`, `h1` 等) |
| T02 | Inter / Noto Sans JP 以外のフォント | テーマの `fontFamily` トークン            |
| T03 | `font-weight: 200` / `100`          | 最低 300 (light)                          |

## スペーシング

| ID  | 禁止                           | 代替                                   |
| --- | ------------------------------ | -------------------------------------- |
| S01 | `margin` / `padding` の奇数 px | 4px グリッド基準 (MUI `spacing()`)     |
| S02 | `gap` の直接 px 指定           | `spacing(1)` = 4px, `spacing(2)` = 8px |

## レイアウト

| ID  | 禁止                                   | 代替                           |
| --- | -------------------------------------- | ------------------------------ |
| L01 | `position: fixed` のフローティング要素 | `Fab` コンポーネントを使用     |
| L02 | サイドバー幅のハードコード             | `LayoutWithSidebar` のテーマ値 |
| L03 | `z-index` の任意値                     | テーマの zIndex トークン       |

## アクセシビリティ

| ID  | 禁止                                   | 代替                                |
| --- | -------------------------------------- | ----------------------------------- |
| A01 | `IconButton` に `aria-label` なし      | 必ず `aria-label` を付与            |
| A02 | 色だけで状態を伝える                   | アイコン・テキストを併用            |
| A03 | `outline: none` でフォーカスリング削除 | `focus-visible` スタイルを使用      |
| A04 | 画像に `alt` なし                      | 装飾画像は `alt=""` + `aria-hidden` |

## AI 生成パターン

| ID   | 禁止                                   | 理由                                   |
| ---- | -------------------------------------- | -------------------------------------- |
| AI01 | カード上部のカラーバー装飾             | AI が生成しやすい無意味な装飾          |
| AI02 | 過度なグラデーション背景               | ブランドトーンと不一致                 |
| AI03 | 丸すぎる角丸 (`rounded-full` on cards) | テーマの `borderRadius` トークンを使用 |
| AI04 | 紫/ピンク系グラデーション              | Kaze ブランドはティール系              |
